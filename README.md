## 🚀 Getting Started

### Prerequisites
```bash
# Python dependencies
pip install -r requirements.txt

# R packages (if using R components)
R -e "install.packages(c('quantmod', 'tidyverse', 'ggplot2'))"
```

### Quick Examples

#### Run Daily Market Analysis
```bash
cd dashboards
python build_data_v2.py
```

#### Generate Daily Quiz
```bash

```
