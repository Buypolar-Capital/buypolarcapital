# Documentation

This folder contains documentation for the Buypolar Capital website.

## Contents

- **PLOTS_GALLERY_README.md** - Instructions for managing the research plots gallery
- **REDESIGN_SUMMARY.md** - Summary of the 2025 website redesign

## Website Structure

```
buypolarcapital/
├── index.html              # Main website file
├── css/                    # Stylesheets
│   ├── base.css           # Base styles & theme variables
│   ├── layout.css         # Header, navigation, grid
│   ├── components.css     # Hero, cards, buttons
│   ├── enhanced-styles.css # Plot cards & modals
│   ├── footer.css         # Footer styles
│   └── forms.css          # Form styles
├── js/                     # JavaScript
│   ├── theme.js           # Dark/light theme manager
│   ├── plots.js           # Plot gallery functionality
│   ├── main.js            # Main site scripts
│   └── ...
├── plots/                  # Research plot PDFs
├── plots_data.json        # Plot metadata
├── assets/                 # Images & static assets
└── docs/                   # Documentation (you are here)
```
